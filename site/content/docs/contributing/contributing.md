Welcome to the `kube-network-policies` community! We are excited to have you contribute.

The project follows the CNCF Code of Conduct and standard Kubernetes contributor guidelines.

## Getting Started

Before your pull requests can be accepted, you must sign the Contributor License Agreement (CLA):

- [Contributor License Agreement](https://git.k8s.io/community/CLA.md)
- [Kubernetes Contributor Guide](https://k8s.dev/guide) - Main contributor documentation.
- [Contributor Cheat Sheet](https://k8s.dev/cheatsheet) - Useful developer resources.

## Contact & SIG Network

We are a sub-project of SIG Network:

- **Slack Channel:** [#sig-network](https://kubernetes.slack.com/messages/sig-network) on Kubernetes Slack
- **Mailing List:** [kubernetes-sig-network](https://groups.google.com/g/kubernetes-sig-network)
